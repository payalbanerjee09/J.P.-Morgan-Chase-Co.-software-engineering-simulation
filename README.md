# J.P. Morgan Chase & Co. - Software Engineering Job Simulation

This repository contains my project implementation and official completion certificate for the J.P. Morgan Chase & Co. Software Engineering simulation completed via Forage.

## 🛠️ Technologies & Tools Used
* **Backend:** Java & Spring Boot (Microservices Architecture)
* **Data Streaming:** Apache Kafka
* **Database:** H2 In-Memory Database
* **API Architecture:** REST APIs & JSON Serialization
